BrowseDNS
BrowseDNS is a public DNS server that enables the Switch and other devices to browse the open Internet using their built-in web browsers. - Terms of Use, FAQ, GitHub

Notice: To exit this browser and connect to the Internet, change your DNS network settings for your device's wifi access point to "Automatic", or to "Manual" with 8.8.8.8 (Google DNS).

To leave user feedback, either join the BrowseDNS Discord or use this form. Thank you!
